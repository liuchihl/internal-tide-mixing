# internal-tide-mixing

Run files and analysis of Oceananigans.jl simulations with internal tide-driven mixing at a sloping rough seafloor. 

initialize_internal_tide.jl: the main model setup (set as a function to be called)
run folder: run files for different cases
postprocess folder: including all analyses and visualization
